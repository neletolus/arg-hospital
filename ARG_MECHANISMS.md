# ARG Hospital - 全仕組み解説書

## 概要

「明日葉内科クリニック」を装った**ARG（代替現実ゲーム）**サイト。一見普通のクリニックWebサイトだが、裏には「プロジェクト明日葉」という人体実験施設の秘密が隠されている。

**技術スタック:** Vue 3 + Pinia + Vue Router + Vite + TypeScript

## 3. ホラー進行システム（`useHorrorProgression.ts` + `stores/horror.ts`）

### 5段階ホラーレベル

| レベル | 発動条件 | 視覚効果 |
|--------|----------|----------|
| **0** | 初期状態 | 通常のクリニックサイト |
| **1** | ページ訪問3回以上 | フッターにカラーシフトアニメーション（30秒周期） |
| **2** | ページ訪問5回以上 or 秘密2個以上発見 | 画像の彩度低下（saturate 0.9）、背景色が暖色寄りに |
| **3** | 隠しページ1つ以上訪問 | 画像のさらなる脱色、ヒーロー部分の明度低下、プライマリカラー変色（#4a7a9b） |
| **4** | 隠しページ3つ以上訪問 | 周期的マイクログリッチ、ヘッダーのちらつき、背景セピア化進行 |
| **5** | ターミナル解放（コナミコマンド） | 深刻な視覚劣化、持続的アーティファクト、全体的に褐色化（#6a5a5a） |

### カラー変遷

```
Level 0: 背景 #f8fafb（明るいグレー）, Primary #2c7bb6
Level 2: 背景 #f6f4f0（暖色系）
Level 3: Primary #4a7a9b（暗い青）
Level 4: 背景 #f0ece4, Primary #5a6a7a（灰色がかった青）
Level 5: 背景 #e8e0d4（セピア調）, Primary #6a5a5a（褐色）
```

### 永続化

- `localStorage`キー: `ashitaba_horror_state`
- 保存内容: 訪問回数、発見した秘密一覧、隠しページ訪問履歴、タイムスタンプ、ターミナル解放状態
- セッションを跨いで進行状態が維持される

## 4. イースターエッグ & 秘密発見トリガー

### 4.1 コナミコマンド（`useEasterEggs.ts`）

- **入力:** ↑↑↓↓←→←→BA
- **効果:** ターミナルアクセス解放、`/terminal`に自動遷移
- **シークレットID:** `konami-code`
- **実装:** 10キーのシーケンスリスナー、間違えるとリセット

### 4.2 ロゴ連続クリック（`useEasterEggs.ts`）

- **トリガー:** クリニックロゴを2秒以内に7回以上クリック
- **効果:** `.logo-flash`アニメーション（invert + hue-rotate）+ 秘密発見
- **シークレットID:** `logo-click`

### 4.3 白文字検出（`useSecretDetection.ts`）

ページ全体のテキスト選択を監視し、以下のキーワードが選択されると発動:

| 隠しキーワード | 意味 |
|----------------|------|
| `被験者の皆様` | 「被験者の皆様」 |
| `逃げてください` | 「逃げてください」 |
| `protocol参照` | プロトコルページへの誘導 |
| `被験者0047` | SUB-0047への言及 |
| `/records` | レコードページへの誘導 |
| `SUB-0023` | 被験者番号 |
| `SUB-0047` | 被験者番号 |

- **シークレットID:** `white-text`
- **仕組み:** `selectionchange`イベントを監視

### 4.4 深夜モード（`useTimeOfDay.ts`）

- **条件:** ローカル時間 0:00〜4:00
- **通常ヒーロー:** 「地域の皆様の健康を支えます」
- **深夜ヒーロー:** 「あなたの身体を、お預かりします」
- **サブタイトル変化:** 「特別プログラムの受付は24時間対応しております」
- **視覚変化:** 背景がダークグラデーション（#1a1a2e → #0d0d1a）に切り替わる
- **シークレットID:** `night-mode`
- **チェック間隔:** 60秒ごと

## 10. 発見パス（プレイヤーの到達ルート）

### ルート1: カジュアル探索
1. 患者口コミの「...続きを読む」を展開 → 不穏なテキスト発見
2. サービスページの白文字を偶然選択 → `/records`や`protocol`の存在を知る
3. URLを直接入力 → 隠しページにアクセス

### ルート2: ソースコード調査
1. F12でDevToolsを開く
2. HTMLコメントから国防生体工学研究所、被験者番号を発見
3. 怪しい経歴項目（国立生体工学研究所）をクリック → コンソールログにヒント
4. `/protocol`、`/records`へ誘導

### ルート3: robots.txt経由
1. 存在しないURLにアクセス → 404ページ表示
2. 404ページのソースに「robots.txtを確認してください」のコメント
3. robots.txtからさらなる手がかり

### ルート4: コナミコマンドマスター解放
1. ↑↑↓↓←→←→BA入力
2. `/terminal`に自動遷移
3. ターミナルから全情報にアクセス可能
4. 即座にホラーレベル5

## 13. 物語テーマ

### 中心設定
一般的な内科クリニックの裏で、遺伝子改変による超人兵士の作成を行う人体実験施設。

### 核心的メッセージ
- 「**被験者の皆様、逃げてください**」（フッター）
- 「**助けて。ここから出して。**」（SUB-0047、複数箇所）

- 「**あなたのアクセスは記録されました**」（訪問追跡テーマ）

### 最大のホラー
適合した被験者（SUB-0023 = 鈴木恵）がスタッフに同化し、一般人と区別がつかないこと。施設は「超人兵士」の作成に成功しており、彼らは既に日常社会に溶け込んでいる。
